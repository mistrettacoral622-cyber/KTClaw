KaiTianClaw 本地设计评审板（全页版）

该版本按「主工作台优先」重建评审结构：借鉴 ClawPort 的布局节奏与操作台氛围，融合 OpenClaw Control Center 的轻治理信息卡，并将首页主色调收敛到你给的 AutoClaw 参考图那种浅暖灰白背景、奶白卡片与橙橘强调。

布局节奏

三段式（左工作对象栏 / 中央主任务区 / 右轻上下文）为主，设置中心改为二级页面体系，避免主工作台被设置流打断。

配色系统

改成 AutoClaw 式浅暖灰白底 + 奶白卡片 + 橙橘色强调；整体更轻、更平，靠局部暖色与圆角卡片建立识别度。

材质 / 卡片层级

统一薄描边（1px）与轻阴影，层级分为：页面底板 / 功能卡片 / 操作焦点卡片，保持治理信息可见但不喧宾夺主。

导航与工作台关系

左侧不是普通路由导航，而是工作对象集合；本轮统一成分身、Channel 频道、任务、团队管理四组折叠栏。首页即主工作台，Settings Center 为次级域。

01 主工作台 / 空态（Default）

基础设置（Basic）

常用配置保持轻表单和卡片节奏，中文优先呈现，保留必要术语（Provider、Workflow、Cron）。

主题与外观

Light Neutral 为默认；浅暖灰白背景、奶白卡片、细描边与橙橘强调保持一致。

语言策略

主界面中文优先，技术关键字与模型名保留英文。

Workflow Presets

预置“评审流”“值守流”“发布流”，支持按 Team 一键应用。

能力与治理（Capability / Governance）

沿用 OpenClaw Control Center 的轻治理风格：状态卡化、口径一致、操作安全等级明确（Conservative / Standard / Full / Custom）。

Memory 策略

定义可写入范围、共享域与保留周期，支持 Team 级继承。

Skills / MCP

统一查看来源、连接状态与权限约束，避免能力散落在多个入口。

权限控制

默认 Standard；高风险命令需二次确认与审计落点。

治理看板

成本趋势、错误率、重试与通道健康度作为治理卡常驻展示。
